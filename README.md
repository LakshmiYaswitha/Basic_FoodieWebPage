#  Foodie - Online Food Delivery Website

Foodie is a simple and attractive front-end food delivery website built using **HTML** and **CSS**.  
It showcases restaurants, food rankings, Delhi food culture, and a pricing table in a clean responsive layout.

---

##  Features

✅ Attractive Header Section  
✅ Hero Banner with Food Image  
✅ Restaurant Cards with Images & Links  
✅ Ordered List of World Famous Foods  
✅ Delhi Food Culture Section  
✅ Styled Food Ranking Table  
✅ Hover Effects on Cards & Table Rows  
✅ Footer Section  

---

##  Technologies Used

- HTML5  
- CSS3
